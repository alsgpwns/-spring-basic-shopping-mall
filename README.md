'배달의 민족' 개발 팀장 김영한 님의 강의입니다.

스프링 예제를 만들어가면서 스프링 핵심 원리를 이해하고, 스프링 기본기를 확실히 다지고자 합니다.

- 스프링의 기본 기능
- 스프링 핵심 원리
- 객체 지향 설계
  
java ↔ OOP ↔ SpringBoot

순수한 자바코드로 예제 애플리케이션을 만들며, 스프링이 왜 만들어졌는지, 핵심은 무엇인지를 깨닫고 이해하고자 합니다.
 
 - SOLID
 - IoC
 - DI
 - Spring Container




강의 목차
1. 객체 지향 설계와 스프링
2. 스프링 핵심 원리 이해1 - 예제 만들기
3. 스프링 핵심 원리 이해2 - 객체 지향 원리 적용
4. 스프링 컨테이너와 스프링 빈
5. 싱글톤 컨테이너
6. 컴포넌트 스캔
7. 의존관계 자동 주입
8. 빈 생명주기 콜백
9. 빈 스코프
